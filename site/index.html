Prepare NEXT without changing LIVE, keep Program visible, route it to several destinations and give speakers, crew and stream operators the views they need. The core workflow stays on your computer and production LAN.

Three-step workflow

From rundown to every screen

01

Build the rundown

Create cues or paste rows from Excel and Google Sheets. Selecting a row prepares NEXT without changing LIVE.

02

Press GO

GO NEXT updates the live cue and timer together. Program remains the operator's source of truth.

03

Route Program

Send the timer, lower third or screen content to several fullscreen, windowed or precisely sized outputs.

Before you download

Questions

Which file should I download?

Apple Silicon Mac users need the ARM64 DMG. Windows 10/11 x64 users should choose the Setup EXE. Do not download GitHub's Source code ZIP or TAR.GZ files unless you are building the app yourself.

Is it really free?

Yes. The MIT License permits use, modification and redistribution. There is no account, subscription, trial limit or output watermark.

Does it need internet?

No for the core app. Phone, backstage, Signal Light and browser views use your local production network. Optional public sharing needs internet.

Why does the operating system warn me?

The public beta is not yet Developer ID notarized or Windows Authenticode signed. Confirm the GitHub source and SHA-256 before overriding a warning.

Can it replace every dedicated timer or media server?

No. It is a software-first option for event teams. Dedicated hardware still wins where physical redundancy is essential, and ProTimer Studio is not a video switcher or full media server.

Language support

English by default. Serbian when you need it.

English and Serbian have full interface coverage. The other 35 language choices cover core operator controls and fall back to English for advanced areas. Every language pack is labeled FULL or CORE in the app; the exact policy is documented in Languages.
